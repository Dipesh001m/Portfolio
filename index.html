    4. End-to-End Real-Time Data Streaming Pipeline with Kinesis, AWS Glue, S3, and Athena

    I developed a data pipeline that begins with data ingestion using Amazon Kinesis, which streams live data for immediate processing. For ETL (Extract, Transform, Load) and data storage, I applied AWS Glue to transform the data, then stored the processed data in Amazon S3, ensuring durability and scalability. To facilitate easy data access, I used AWS Glue Crawler to automate data cataloging, which maintains an up-to-date catalog of available datasets. Finally, for querying, I leveraged Amazon Athena, a serverless solution that provides fast, efficient querying of the data stored in S3 without the need for infrastructure management.
